# BitOS GNU/Linux

BitOS is a rolling-release GNU/Linux distribution based on Kali Linux,
focused on cybersecurity, penetration testing, and red team operations.

## Features
- Rolling release
- Preinstalled pentesting tools
- Custom BitOS branding
- Optimized for security professionals

## Project Goals
- Provide a clean hacking-focused OS
- Custom installer and branding
- Community-driven development
